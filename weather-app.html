Weather App

Home

Description:

A clean and user-friendly weather application that allows users to view real-time weather data for any city around the world. It integrates the OpenWeatherMap API to fetch data like temperature, weather condition, humidity, and wind speed.

Features:

Real-time weather updates

City search functionality

Weather icons for visual feedback

Mobile responsive design

Error handling for invalid city names

Tech Used: HTML, CSS, JavaScript, API
